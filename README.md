---

## ‚ú® Key Features

### üó∫Ô∏è Expedition Management
* **Dynamic Itineraries:** Automated day-by-day route mapping with key highlights for high-altitude passes.
* **Interactive Stats:** Real-time visualisation of distance (km), duration, and rider count using custom capsule UI components.
* **Difficulty Tiering:** Categorization of rides into *Beginner*, *Intermediate*, and *Advanced* based on terrain complexity.
* **Essentials Checklist:** Context-aware document reminders for Inner Line Permits (ILP), RC, and insurance.
* **Medical Preparedness:** Specialised gear checklists including portable oxygen and AMS medications.

### üí° Rider Advice & Community Intelligence
> The "Advice" module is designed to provide critical survival and performance tips for extreme riding conditions.

* **Route and Path Tips:** Community-provided guides on handling all types of terrain and routes.
* **Mechanical Tips:** Quick-fix guides for common mid-ride issues like chain tightening and cold-start troubleshooting.
* **Packing Logic:** Community-vetted packing lists for different climates (Coastal vs. Himalayan).
* **Road Intelligence:** Updates on pass openings (Sach Pass, Baralacha La) and fuel station availability.
